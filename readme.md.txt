# Tercer desafío de Backend

## Introduccion

Tercer desafío de `React JS`, Espero les guste :)

La idea de éste desafío es abrir un servidor express en el puerto 4000, y mediante la url, si se introduce `/products`: que muestre todos los productos agregados, y si se usa el query /products`?limits=numero` se limite la cantidad de productos que se muestran en pantalla, y por último si se utiliza `/products/id` se va a mostrar el producto con su id introducido. :)
